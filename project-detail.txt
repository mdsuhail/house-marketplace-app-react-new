- Setup github and install React app
	npx create-react-app house-marketplace-app-react-new --use-npm
	cd house-marketplace-app-react-new
	
	git remote add origin https://github.com/mdsuhail/house-marketplace-app-react-new.git
	git branch -M main
	git push -u origin main

- Create config file firebase.config.js in project src folder and add the code provided from firebase.

- Setup the Authentication provider in Firebase console for Email/Password and Google

- Setup the API key in GCP(Google Console) for Geo Location api. (Note- Api did not work becuase of the billing issue)
